# Enterprise Features

Lantern Enterprise adds compliance, security, and team management features.

## License

## Features

### PII Detection
Scan all prompt and response content for PII (emails, phone numbers, SSNs, credit cards). Flag or redact automatically.

### SOC2/HIPAA Audit Export
Export audit-ready logs in formats required by SOC2 and HIPAA compliance frameworks.

### Alerting (Slack, PagerDuty, Webhooks)
Get notified when quality scores drop, costs spike, or errors increase.

### Team-Scoped RBAC
Restrict trace visibility by team. Each team only sees traces from their agents.

### Managed Cloud
Skip self-hosting. Use Lantern's managed cloud ingest endpoint with guaranteed uptime and SLA.
